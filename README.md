# Multitool

A collection of functions useful for programming challenges and beyond

## Installation

This package can be installed by adding `multitool` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:multitool, "~> 0.3.1"}
  ]
end
```

Then, update your dependencies:

```elixir
mix deps.get
```
