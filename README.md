## Installation

This package can be installed by adding `multitool` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:multitool, "~> 0.3.4"}
  ]
end
```

Then, update your dependencies:

```elixir
mix deps.get
```
